# KDE Theme Changer
Поддерживается только системы Linux.

Установка и запуск
```
git clone https://github.com/S-Semyon/Quasar-Tweaker.git
cd "Quasar-Tweaker"

# Arch
pacman -S python-pyqt5

# Ubuntu
apt-get install python3-pyqt5

# Fedora
dnf install python3-qt5.x86_64

# Запуск
python3 main.py 
```
